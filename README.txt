[ The project description ]    
The project in this repository is data analysis of population growth in the past. 
It uses the CSV datafile "file-population.csv" that contains historical population data between the years 1960 and 2021. 
The data source is World Bank, downloaded on 2023/03/01. 
The dataset includes all of the countries in the world, different regions, and differrent income groups. 

Currently the Jupyter notebook file only contains the basic analysis of the world total population, but the purposes of the project is to analyze the population growth in different countries and income groups. 


[How to install it]
Install this project by downloading the CSV datafile "file-population.csv" and jupyter notebook file "CAS-project-cleancode.ipynb". 
You need to launch jupyter notebook from Terminal
